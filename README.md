# Room Classification v1

Классификация помещений и смежных зон на фотографиях недвижимости с помощью модели **YOLO**.

Модель определяет тип комнаты или пространства на изображении и относит его к одному из **7 классов**.

## Классы

| Класс | Описание |
|-------|----------|
| `bathroom` | Ванная комната, санузел |
| `bedroom` | Спальня |
| `kitchen` | Кухня |
| `other` | Прочие помещения (гостиная, коридор, кабинет и т.д.) |
| `pool` | Бассейн |
| `view_from_window` | Вид из окна |
| `yard` | Двор, участок, внешняя территория |

## Задача

По входному изображению модель предсказывает класс помещения/зоны. Проект ориентирован на автоматическую разметку и анализ фотографий объектов недвижимости: фильтрация, группировка снимков по типу комнаты, контроль качества контента.

## Технологии

- **YOLO** — детекция и классификация объектов на изображении
- Python
